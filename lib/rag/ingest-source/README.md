Сюда владелец продукта кладёт материалы для базы знаний — только те, на
которые есть права (собственные тексты, конспекты своими словами, public
domain, лицензированный контент). Не загружать оцифрованные коммерческие
книги целиком.

Структура: один `.md`-файл на источник, в подпапке с именем `kb_namespace`
модуля (например `burnout/`, `self_esteem/`). Запуск ингеста: `bun run rag:ingest`.
