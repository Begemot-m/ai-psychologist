# AI-психолог — Telegram Mini App (прототип)

Бережное психологическое сопровождение в Telegram: для людей в терапии и для тех, кому нужна
бытовая психологическая поддержка. Инструмент разработан с участием психологов, принцип —
«не навреди себе и окружающим». Не заменяет работу со специалистом, а сопровождает.

Отличия от «голого чат-бота»: встроенный кризисный слой безопасности, подбор подхода и
тематических модулей под пользователя, RAG по проверенным материалам, премиальный интерфейс.

Полные технические решения — в [ARCHITECTURE.md](./ARCHITECTURE.md).

## Definition of Done

- мини-апп открывается из Telegram, пользователь сохраняется в БД;
- онбординг → рекомендация → выбор модуля и подхода работают;
- чат отвечает с RAG-контекстом; кризисный классификатор срабатывает на тестовых фразах и
  выдаёт протокол;
- free-лимит упирается в пейволл, оплата Stars активирует paid и переключает модель на Sonnet;
- репозиторий запускается по README; код в git и запушен.

## НЕ делать в прототипе

- админ-панель, аналитику, A/B-тесты;
- все модули кроме двух (остальные — заглушки в UI);
- TON/крипто, реферальные системы, рекламу;
- авторизацию помимо Telegram `initData`;
- сложную память/долгосрочный профиль сверх суммаризации сессии;
- мобильные нативные сборки.

## Открытые вопросы и блокеры

- **Учётные данные**: Anthropic API key, Supabase-проект, Telegram bot token (через @BotFather) —
  ничего не создано. В `.env.example` — только плейсхолдеры. Перед Фазой 1 нужен Supabase-проект
  и его ключи; перед Фазой 2/5 — бот и токен.
- **Кризисные контакты** (`crisis_resources`): в конфиге — плейсхолдеры, реальные номера горячих
  линий владелец вписывает сам из официальных источников. Не выдумывать.
- **База знаний для RAG**: загружать можно только материалы, на которые есть права (свои тексты,
  конспекты своими словами, public domain, лицензированный контент). Не оцифровывать и не грузить
  коммерческие книги целиком.
- **Эмбеддинги**: выбран Voyage AI как бесплатное временное решение (см. ARCHITECTURE.md) —
  при необходимости заменить провайдера позже.
